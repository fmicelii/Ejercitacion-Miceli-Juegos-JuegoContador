## 🚀 Inicio Rápido

1. **Navegar a la carpeta del juego:**
   ```powershell
   cd juego-contador
   ```

2. **Instalar dependencias:**
   ```powershell
   npm install
   ```

3. **Iniciar la aplicación:**
   ```powershell
   npm start
   ```

4. Abrir `http://localhost:3000` en tu navegador
